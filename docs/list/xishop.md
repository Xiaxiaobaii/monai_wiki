# 汐店
> 汐店系列操作详细介绍

## 功能列表

#### 鱼
鱼的出现时间目前由天气决定，部分鱼只会在特定时间出现，鱼有6种稀有度等级分别从0-5划分，等级越高难度越高，同时对应价格和稀有程度也越高，相同稀有度的鱼价格根据种类也不相同

#### 钓鱼
Usage: 钓鱼 (Class)

介绍：钓鱼，消耗鱼饵，Class参数为鱼饵种类，不同的鱼饵有不同的特性与价格，默认使用初级诱饵，当背包不存在指定诱饵且汐店有库存时会自动购买

示例：钓鱼 高级诱饵

#### 汐店
Usage: 汐店 (Class) (num)

介绍：汐店，常规鱼饵购买位置，无参数时会显示当前可购买的鱼饵列表，Class为要购买的鱼饵名字，num为购买数量，默认为1

#### 卖鱼
Usage: 卖鱼 (Class) (num)

介绍：卖鱼，可以卖掉背包中的鱼，Class为鱼饵名称，num为卖出数量
